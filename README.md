## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. **Clone the repository:**
```bash
git clone <repository-url>
cd hirewise-frontend
```

2. **Install dependencies:**
```bash
npm install
```

3. **Set up environment variables:**
Create a `.env` file:
```env
VITE_BACKEND_URL=http://localhost:5000
```

For production:
```env
VITE_BACKEND_URL=https://hirewise.up.railway.app
```

4. **Start development server:**
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173)

5. **Build for production:**
```bash
npm run build
```

## ğŸ” Authentication Flow

1. **Login/Signup**: User submits credentials
2. **Token Storage**: JWT stored in localStorage
3. **API Requests**: Token auto-injected via axios interceptor
4. **Protected Routes**: `isAuthenticated()` check
5. **Logout**: Clear localStorage and redirect
